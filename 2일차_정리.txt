 Pytest

Pytest는 Python 코드를 테스트하기 위한 강력하고 유연한 프레임워크입니다.
단위 테스트와 통합 테스트를 지원하며 간단한 문법으로 테스트를 작성하고 실행할 수 있습니다.

Pytest는 자동으로 파일 이름이 test_*.py 또는 *_test.py, 함수 이름이 test_로 시작
클래스 이름이 Test로 시작하는 항목을 찾습니다.

실행방법은 pytest만 치거나 항목 이름을 적어서 특정항목만 실행 할 수 있습니다.


 Coverage

 Coverage는 실제로 전체의 코드중의 몇개의 코드가 한번이라도 사용되는지 보여준다


테스트 실행 자동화 스크립트

black → 코드 포맷팅
ruff → import 정리 + 린트 검사
mypy → 타입 검사
pytest + coverage → 테스트 실행 + 커버리지 리포트


GitHub Actions

GitHub Actions는 GitHub에서 제공하는 자동화 플랫폼이다.
코드를 push하거나 Pull Request를 만들면, 정해둔 스크립트가 자동으로 실행된다.
이걸 이용해서 github에 push 할 때 마다 검사를 자동으로 실행 되도록 할 수 있다
