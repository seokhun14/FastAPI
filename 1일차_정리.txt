1. Meetifyr
여러 사람의 약속/일정 조율 도구.

팀이나 친구들 일정 맞출 때 유용

온라인에서 시간 선택, 투표 같은 기능 제공

2. Poetry
파이썬 패키지/가상환경 관리 도구.

requirements.txt 대신 pyproject.toml 사용

의존성 관리 + 빌드/배포까지 지원

3. Black
파이썬 코드 포매터 자동 정렬기

자동으로 코드 스타일 통일

4. Ruff
 파이썬 linter + 포매터.

flake8, isort, pylint, mypy 일부 기능까지 통합



5. mypy
파이썬 정적 타입 검사기.

실행 전 type hint 기반으로 타입 오류 체크

동적 언어인 파이썬 코드를 정적 언어처럼 안정성 확보 가능
